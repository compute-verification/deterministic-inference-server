How it works

A gateway signs your prompt and sends it through a passive tap to the language model. The tap simultaneously copies the prompt and the response to a verifier, which re-runs the same inference and compares the output. Same prompt · same seed · deterministic config ⇒ bit-identical tokens. If they differ, the verifier raises an alarm.

Model: Qwen3-1.7B on a single H100. Determinism: enforce_eager, FLASH_ATTN, batch invariance, seed=42, temperature=0.
